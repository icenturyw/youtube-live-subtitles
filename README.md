# YouTube 实时字幕生成器

一个 Chrome 扩展，为没有字幕的 YouTube 视频实时生成字幕。

## 功能特点

- 🎤 **实时语音识别** - 使用 Web Speech API 进行实时语音转文字
- 🌍 **多语言支持** - 支持中文、英语、日语、韩语等多种语言
- ⚡ **低延迟** - 本地处理，几乎无延迟
- 🎨 **可自定义** - 可调整字幕大小、位置等
- 🔒 **隐私安全** - 所有处理在本地完成，不上传数据

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `youtube-live-subtitles` 文件夹
5. 完成！扩展图标会出现在浏览器工具栏

## 使用方法

1. 打开任意 YouTube 视频页面
2. 点击浏览器工具栏中的扩展图标
3. 选择识别语言（默认中文）
4. 点击 **开始生成字幕** 按钮
5. 首次使用会请求麦克风权限，请点击允许
6. 字幕会实时显示在视频底部

## 设置选项

| 设置 | 说明 |
|------|------|
| 识别语言 | 选择视频音频的语言 |
| 字幕大小 | 调整字幕字体大小 (16-36px) |
| 字幕位置 | 底部或顶部 |
| 显示实时识别 | 是否显示未确认的文字（斜体显示） |

## 工作原理

1. **音频捕获**: 通过浏览器的 Web Audio API 捕获系统音频
2. **语音识别**: 使用 Chrome 内置的 Web Speech API 进行语音转文字
3. **字幕渲染**: 将识别结果实时渲染到视频播放器上方

## 注意事项

- ⚠️ 需要使用 **Chrome 浏览器**（其他浏览器可能不支持 Web Speech API）
- ⚠️ 需要允许 **麦克风权限**（用于音频捕获）
- ⚠️ 识别准确度取决于音频质量和语音清晰度
- ⚠️ 背景音乐可能会影响识别效果

## 技术栈

- Chrome Extension Manifest V3
- Web Speech API
- Web Audio API
- Vanilla JavaScript
